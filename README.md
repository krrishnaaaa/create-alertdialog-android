# AlertDialog in Android

Source code for the blog post: [AlertDialog in Android — The Complete Guide](https://pcsalt.com/android/alertdialog-android-complete-guide/)

## What's covered

- Basic AlertDialog with positive, negative, and neutral buttons
- AlertDialog with icon
- Multi-choice selection dialog (checkboxes)
- Single-choice selection dialog (radio buttons)
- Custom layout using XML (Material TextInputLayout login form)
- Custom layout built programmatically
- Button color customization
- Override button click behavior (validation before dismiss)

## Tech stack

- Kotlin
- Material 3
- ViewBinding
- minSdk 34, targetSdk 36
